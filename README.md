# Composites Manufacturing

A single-page Joby-inspired title page. The button is, regrettably, unclickable.

## Hosting on GitHub Pages

1. Create a new public repository on GitHub.
2. Upload `index.html` to the root.
3. Go to **Settings → Pages**, set Source to "Deploy from a branch," Branch to `main` / `(root)`, save.
4. ~1 minute later, the page goes live at `https://<your-username>.github.io/<repo-name>/`.

## Image

The background photo is a Joby press image of the S4 over San Francisco Bay near the Golden Gate Bridge, hotlinked from Joby's CDN. If you'd rather bundle it locally, download it, place it in an `images/` folder, and replace the `src` URL in `index.html` with `images/your-filename.jpg`.
